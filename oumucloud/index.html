欧穆云软件科技

认识 OuMuCloud

徐州欧穆云软件科技合伙企业（普通合伙）提供全面的质优价廉的专业软件外包服务，企业技术栈丰富，适用于各种业务场景（网站、安卓 APP、微信小程序、桌面应用、爬虫等），我们有充分的实力与信心实现客户需求。

企业所有技术人员均为全栈工程师，技术水平成熟，拥有丰富的项目开发、维护、团队协作经验，确保高质量完成项目并快速交付。

客户可以在我们能力范围内自由选择技术栈进行项目开发：对于 APP，可以选择 Flutter 或 uni-app 进行跨平台开发（Android 和 iOS），或使用 Kotlin 与 Swift 分别进行原生开发；对于桌面应用，可使用 Electron 或 Qt 快速开发跨平台（Windows、macOS、Linux）应用；对于小程序，通常用 uni-app 多端平台快速部署；对于网站，支持客户自由选择前后端技术栈、数据库与 UI 框架，目前前后端框架支持 Vue3 + Spring Boot/Fastify.js，数据库支持 MySQL、PostgreSQL，CSS 框架支持 Bootstrap、Layui、Tailwind、Element UI。

欧穆云不仅提供软件定制与外包服务，还有更多扩展业务可供选择：UI 设计——助力客户构建精美的现代化 UI 界面；Web 爬虫——爬取海量资源与数据；云服务——高配置低价格服务器、虚拟主机等云产品，网站托管、云盘存储等服务。

我们坚守"客户至上"的原则，以客户利益为根本出发点和落脚点，全心全意为客户服务，始终把客户利益放在第一位，尽全力确保快速无误地交付令客户满意的高质量项目。

我们的产品

欧穆云目前主营 TradingGML 与 AOE 莹轩。我们不仅在功能和界面上追求简洁与高效，更在技术创新上不断求索。采用先进的计算技术、算法和大数据分析，对产品性能进行持续优化，确保在处理大量数据和复杂任务时依然能保持轻盈和快速响应。

TradingGML
  • 支持伪造本地通话记录与短信信息
  • 提供财务波动率分析、永久投资组合调仓辅助
  • 内置 CFD 收益、利率转换、ETF 补仓等计算工具
  • 提供密码记录器、Markdown 笔记本功能
  • 支持导入、导出 CSV 或 JSON 格式的备份
AOE 莹轩
  • 用 Java + libGDX 开发的 2D 安卓游戏
  • 玩家通过经济管理、科技研发和军事扩张来发展国家
  • 贯穿世界历史，拥有海量剧本与国家可供选择
  • 可自由修改并分发，目前仅适配安卓客户端
